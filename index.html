The Circuit

Simple language-agnostic cluster programming

Download this project as a .zip file Download this project as a tar.gz file

Install

$ go get github.com/gocircuit/circuit/cmd/circuit

Run

To run the first circuit agent, pick a network address and a local mount-directory, then run

circuit -a 10.20.30.7:11022 -m /circuit

The circuit agent will respond by printing out its “circuit URL”, something like

…
circuit://10.20.30.7:77/78517/R56e7a2a0d47a7b5d
…

Copy the circuit URL, so you can point the next circuit agent to it

circuit -a 10.20.30.5:77 -m /circuit -j circuit://10.20.30.7:77/78517/R56e7a2a0d47a7b5d

Explore

Go the local circuit mount-point and look around. All directories include readable help files.

cd /circuit
ls
